Comparative Analysis

Why Multi-Agent Ensemble is Better

1

Better Calibrated Confidence

Single agent shows overconfidence (93.3%) while multi-agent provides more calibrated estimates (86.7%). In Case 0, single agent was 95% confident in a WRONG verdict, while multi-agent correctly identified ambiguity with appropriate uncertainty.

2

Detects More Mutation Types

Multi-agent ensemble identified 8 mutation types vs only 3 for single agent. The adversarial debate structure forces deeper analysis, uncovering subtle distortions like "missing context" and "numerical framing shifts" that single agent missed.

3

Transparent Reasoning Chain

Multi-agent provides full debate transcript showing how agents argued, conceded points, and reached consensus. This transparency enables human oversight and audit trails - critical for fact-checking applications.

4

Handles Ambiguity Appropriately

Single agent forced binary verdicts (2 faithful, 1 mutated). Multi-agent correctly identified Case 0 as genuinely ambiguous - neither clearly faithful nor clearly mutated. This nuance is essential for real-world fact-checking.

5

Adversarial Testing Reduces Blind Spots

Prosecutor and Defense agents attack each other's arguments, exposing weaknesses. In Case 2, the Defense conceded the numerical inflation point, while Prosecutor acknowledged temporal paraphrasing was acceptable - balanced reasoning single agent cannot achieve.

Why Multi-Agent is Better Here
Single agent missed the subtle but important shift from "more than 14,500" (lower bound) to "less than 14,550" (upper bound). This framing change subtly caps the death toll rather than emphasizing minimum. Multi-agent's Prosecutor caught this, and after 4 rounds of debate, the ensemble correctly concluded: genuinely ambiguous - a more honest assessment.

Case 2 Badmotorfinger - Sales Figures

Truth
"Badmotorfinger was certified double platinum by the RIAA in April 1996, signifying two million shipped copies; however, only 1.5 million copies had been sold by October 2010."
Single Agent MUTATED

date distortion numerical exaggeration temporal misplacement
Multi-Agent MUTATED
Confidence: 90%
temporal mismatch (low) numerical distortion (high) missing context (medium)
Refined Severity Assessment
Both agreed MUTATED, but multi-agent provided severity gradation: "after March 1996" is LOW severity (acceptable paraphrase), while "1.8 million sold" is HIGH severity (contradicts 1.5M in truth). Defense explicitly conceded the numerical inflation: "constitutes an undeniable factual distortion."
Key Concession Moment
Round 3
DEFENSE CONCEDES "We concede this point to the Prosecutor. The claim's figure of 'more than 1.8 million copies sold' lacks explicit sourcing and contradicts the original fact's '1.5 million copies sold.' This numerical inflation constitutes a high-severity distortion."

Conclusion

Multi-agent ensemble significantly outperforms single-agent approaches for fact verification by providing calibrated confidence, comprehensive mutation detection, and transparent reasoning. The adversarial debate mechanism catches subtle distortions that single-agent approaches miss, while the consensus process handles genuine ambiguity appropriately.
